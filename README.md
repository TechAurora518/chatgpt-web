# ChatGPT Bot Web

This is a ChatGPT web application demo with express and vue3

![cover](./docs/cover.png)
## Intro
1. `node >= 18`
1. `pnpm install` node deps
2. set `OPENAI_API_KEY` in `.env`

## Run
```shell
# 1. run service
pnpm run service

# 2. run web
pnpm run dev
```
